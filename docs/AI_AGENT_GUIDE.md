# AI Agent Guide

## Как работать с кодом

1. Не нарушать слои архитектуры
2. Не вызывать API напрямую из ViewModel
3. Использовать зависимости через swift-dependencies

## Как добавлять новую фичу

1. Создать View
2. Создать ViewModel
3. Добавить Repository
4. Добавить Service (если нужен)
5. Зарегистрировать dependency

## Что нельзя делать

- Жёстко связывать компоненты
- Использовать глобальное состояние
- Хранить бизнес-логику во View

## Что важно

- Код должен быть тестируемым
- Все зависимости должны мокаться
