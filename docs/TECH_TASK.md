Разработка MVP приложения «Точка входа»
Постановка задачи
Необходимо разработать минимально жизнеспособный прототип (MVP)
приложения, которое заменяет пластиковую карту сотрудника и позволяет использовать цифровой пропуск для входа в здание.
MVP должен выполнять одну ключевую функцию: генерировать цифровой пропуск сотрудника в виде QR-кода, который можно использовать для идентификации на проходной.
На данном этапе интеграция с корпоративными системами не требуется. Основная цель — продемонстрировать концепцию цифрового пропуска и пользовательский сценарий его использования.
